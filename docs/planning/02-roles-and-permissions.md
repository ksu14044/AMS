# 02. 사용자 역할 및 권한 체계

## 역할 목록

| 그룹 | 역할 | 설명 |
|------|------|------|
| 관리자 | 원장·학원 관계자 | 전체 관리, 수강반 배정, **조회 시작일** 설정 |
| 선생님 | 국어/영어/수학 담임 | 담임 반 운영, **수업기록** 작성 |
| 선생님 | 행정 | 학원 전체 보조, **학원 공지** ([DECISIONS.md](./DECISIONS.md) §2) |
| 조교 | 국어/영어/수학 조교 | 담당 반 운영, 정답지·채점·클리닉 결과 |
| 학생 | 학생 | 수강 반 조회·클리닉 예약·지정 영상 인증 |
| **학부모** | **학부모** | **v3.0** — 자녀 학습 **읽기 전용** ([§19](./DECISIONS.md#19-학부모-계정)) |

교직원: **역할 7종** → 원장 승인.  
학생·학부모: **별도 가입** → 원장 승인.

`ACADEMY_ADMIN` — 7종 외.

## 관리자

- 학생·반·교직원·**학부모** 관리
- 수강반 배정 + **`accessible_from`** 설정
- **학부모–학생 연결**

## 담임 선생님

- **수업기록**·콘텐츠 운영, 조교 반 지정
- 클리닉·**결과 프리셋** 관리
- 보고서 생성·코멘트, **학부모–학생 연결**

## 조교

- 담당 반 **수업기록** 하위 CRUD·**정오표·점수 입력**
- **반 전체 정답지 조회**, 클리닉 **프리셋 결과** 입력
- **학부모–학생 연결**

## 학생

- `accessible_from` 이후 **수업기록**만 조회 ([§15](./DECISIONS.md#15-학생-조회-시작일))
- 홈: **미완료** 숙제·테스트·클리닉
- **지정 영상**만 인증; 알림 **완료 시** 소멸 ([§16](./DECISIONS.md#16-알림-완료-기준))

## 학부모 (v3.0)

- 학생과 **동일 가입·승인**
- 교직원이 **자녀 연결** 후 읽기 전용 조회
- 쓰기 API 없음

## RBAC

```
권한 = f(역할, academy_id, class_id, enrollment | assignment | parent_link)
```

- 학부모: `parent_student_link` → 자녀 `student_id` 위임 **GET만**

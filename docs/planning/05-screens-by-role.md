# 05. 역할별 화면 설계

## 5.1 선생님(담임)

### 반 상세 탭 (v3.0)

| 탭 | 기능 |
|----|------|
| **수업기록** | 날짜별 수업·요약; 숙제·테스트·클리닉·영상 |
| 수업 정보 / 공지 / 교재 | 수업기록과 **별도** |
| 공부기록 | 기간·프리셋 |
| 보고서 | 기간·프리셋·**학생 선택·일괄** 생성 |
| 학생 목록 | **조회 시작일** 설정 |

## 5.2 조교

- 수업기록 하위 **정오표·점수** 입력
- **정답지 조회**, 클리닉 **프리셋**·결과 입력
- 학부모–학생 연결

## 5.3 학생

### 홈 (v3.0)

- 🔔 **미완료** 알림
- **미완료 숙제·테스트·클리닉** 카드
- 수업기록 기반 반 홈

### 알림

- `[ 전체 ]` `[ 미완료 ]`

## 5.4 학부모 (v3.0)

- 자녀 선택 → 미완료·공부기록·보고서 **읽기 전용**

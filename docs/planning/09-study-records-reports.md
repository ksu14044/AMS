# 09. 공부기록 및 성실도 보고서

## 11-1. 저장 원칙

- 학생 활동 **실시간 기록·영구 보존**
- 삭제: 관리자만
- 반 삭제 시 기록은 **아카이브** 이전

## 집계 항목·가중치 (원본 §11-1)

| 항목 | 저장 데이터 | 가중치 |
|------|-------------|--------|
| 숙제 | 제출 여부, 제출 일시, 결과(점수/등급) | 40% |
| 클리닉 | 예약 여부, 출석 여부, 담당 조교, 내용 메모 | 30% |
| 테스트 | 응시 여부, 점수, 등급, 반 평균·상위 %; 게이지용 [percentile_rank](./DECISIONS.md#9-테스트-점수--종합-게이지-30-환산) | 30% |
| 영상 인증 | 인증사진 제출 여부, 제출 일시, 영상명 | 별도 %·보고서 항목 (종합 게이지 가산 없음 — [DECISIONS.md](./DECISIONS.md) §7) |

## 게이지 색상

| 범위 | 색 | 의미 |
|------|-----|------|
| 75% 이상 | 🟢 초록 | 우수 — 숙제·클리닉·테스트 성실히 이행 |
| 50–74% | 🟡 주황 | 보통 — 일부 항목 보완 필요 |
| 50% 미만 | 🔴 빨강 | 미달 — 담임 선생님 면담 권고 |

학생 반 홈·공부기록: `75% 이상 → 초록 / 50-74% → 주황 / 50% 미만 → 빨강`  
종합 달성률 예: **72% — 잘 하고 있어요! 🟢**

## 11-3. 시험 후 성실도 보고서

**트리거:** 테스트 결과 입력 시

**구간:** 이전 테스트일 ~ 이번 테스트일

**포함 항목**

- 보고서 기간
- 숙제 성실도 (%)
- 클리닉 성실도 (%)
- 테스트 결과 (점수, 반 평균, 상위 %)
- 영상 인증 현황
- 종합 점수 (숙제 40% + 클리닉 30% + 테스트 30%만; 영상 제외)
- 등급 A(90+) / B(75+) / C(60+) / D(50+) / F
- 담임 코멘트 (수동 입력)

**산출:** PDF 다운로드

### 보고서 예시 (원본 §11-3)

국어 남고1반 / 김민준 / 5월 3주차 문학 테스트 보고서  
보고서 기간: 2025.04.28 ~ 2025.05.21

| 항목 | 수치 | 달성률 | 등급 |
|------|------|--------|------|
| 숙제 제출 | 7 / 8건 | 87.5% | A |
| 클리닉 출석 | 6 / 8회 | 75.0% | B |
| 테스트 점수 | 84점 / 반 평균 78점 | 상위 23% | B |
| 영상 인증 | 4 / 5편 | 80.0% | A |

- **종합 성실도 82점 — B등급** (숙제×0.4 + 클리닉×0.3 + **이번 시험 원점수 %**×0.3; 테스트 행의 상위 %는 표시용)
- 담임 코멘트 예: 「클리닉 출석이 조금 아쉽지만 전반적으로 성실합니다…」

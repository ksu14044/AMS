# 08. 클리닉 예약 규칙

| 규칙 | 내용 |
|------|------|
| 예약 기간 | 월 00:00 ~ 토 23:00 (해당 주, 변경 가능 기간) |
| 학생 신청 요일 | **월~금만** (원본 §2-4) — 토·일 슬롯 없음 |
| 잠금 | 토 23:00 서버(Asia/Seoul) 자동 잠금, 학생 변경 불가 |
| 슬롯 | 담임이 요일·시간·조교 사전 설정 (같은 시각·다른 조교 = 슬롯 별도) |
| 학생 예약 | **같은 반·같은 주·같은 요일·같은 시각**에는 1건만 (조교 무관) |
| 조교 | 슬롯당 1명, 학생은 조회만 |
| 결과 | 조교 또는 담임이 출석·메모 입력 |
| 알림 | 잠금 시 확정, 슬롯 변경 시 개별 알림 |
| 신규 주차 | 월 00:00 다음 주 슬롯 자동 생성 |

## Cron 작업

| 스케줄 | 작업 |
|--------|------|
| `0 0 * * 1` | 다음 주 슬롯 템플릿 생성 |
| `0 23 * * 6` | 해당 주 예약 LOCK + 확정 알림 |

## 데이터

- `clinic_week` (반, 주차 시작일, status)
- `clinic_slot` (요일, 시간, assistant_id)
- `clinic_reservation` (slot_id, student_id, status)

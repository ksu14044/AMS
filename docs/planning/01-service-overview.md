# 01. 서비스 개요 및 목적

## 한 줄 정의

클래스팅·아이엠스쿨 등을 참고하되, **입시 학원 특화**(클리닉 예약, 수업기록·정오표 채점, 재시험, 성실도 보고서, 영상 인증, 학부모 조회)를 갖춘 **반(Class) 기반** 통합 관리 앱.

## 핵심 과목

- 국어 / 수학 / 영어

## 반별 관리 항목 (v3.0)

| 구분 | 항목 |
|------|------|
| **수업기록** (집계 단위) | 수업일·요약 + 귀속 숙제·테스트·클리닉·영상 |
| **별도 섹션** | 공지 · 수업 정보 · 교재 |

## v3.0 주요 변경 (2026-05-30)

- **수업기록 탭** — 날짜별 수업 단위로 숙제·테스트·클리닉·영상 귀속
- **정오표 수동 채점** — OCR 없음; 교직원이 정답지·학생 답안 입력
- **마감 없음** — 점수 입력 = 완료; D-1 알림 폐기
- **테스트 석차** — 상위 % 대신 등수; 재시험 최대 3회
- **클리닉 결과 프리셋** — 양식 저장·재사용
- **영상 인증** — 지정 학생만 (기본 미선택)
- **보고서** — 기간·프리셋·학생 선택 수동 생성
- **학부모** — 가입·승인 후 자녀 연결, 학습 현황 조회

상세: [DECISIONS.md](./DECISIONS.md) §10~§19.

## 대상 플랫폼

- 웹 + 모바일 반응형

## 참고 플랫폼

클래스팅, 아이엠스쿨, Band, 수능완성 플랫폼, Classting

## 성공 지표(구현 시 참고)

- 담임·조교가 **수업기록** 중심으로 반 콘텐츠 운영
- 학생·학부모가 미완료 숙제·테스트·클리닉을 한눈에 파악
- 교직원 정오표 입력 → 점수·석차·재시험·보고서 자동 연동

# 01. 서비스 개요 및 목적

## 한 줄 정의

클래스팅·아이엠스쿨 등을 참고하되, **입시 학원 특화**(클리닉 예약, 시험 후 성실도 보고서, 영상 인증)를 갖춘 **반(Class) 기반** 통합 관리 앱.

## 핵심 과목

- 국어 / 수학 / 영어

## 반별 관리 항목

공지 · 영상 · 숙제 · 테스트 · 클리닉 (+ 수업 정보, 교재)

## 대상 플랫폼

- 웹 + 모바일 반응형

## 참고 플랫폼

클래스팅, 아이엠스쿨, Band, 수능완성 플랫폼, Classting

## 성공 지표(구현 시 참고)

- 담임·조교가 반 콘텐츠를 한 화면(반 상세)에서 운영 가능
- 학생이 배정된 반만 조회, 클리닉·인증·공부기록이 한 흐름으로 연결
- 테스트 입력 후 성실도 보고서가 자동 생성

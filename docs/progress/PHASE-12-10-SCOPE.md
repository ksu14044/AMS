## 12.10-1 · 정답지 업로드

| 항목 | 규칙 |
|------|------|
| 저장 | 업로드 파일 **원본 그대로** (PDF 변환 없음) |
| 허용 | `pdf`, `png`, `jpg`, `jpeg`, `doc`, `docx`, `hwp`, `hwpx` |
| DB | 기존 `answer_key_pdf_path` 컬럼에 상대 경로 저장 (마이그레이션 없음) |
| 기존 파일 | 이미 저장된 `.pdf` 경로 유지 |
| 배포 | Docker `app` 사용자 `/data/uploads` 쓰기 권한 |

---

## 12.10-2 · 완료 처리 · 목록 정렬

| 항목 | 규칙 |
|------|------|
| 미완료 | 대상 학생 중 **선생님이 결과 미입력** |
| 숙제·테스트 | 「완료 처리」 버튼 **제거** |
| 자동 완료 | 대상 **전원** 결과 입력 시 `COMPLETED` (테스트는 석차·반평균 포함) |
| 확인 탭 정렬 | **미입력 우선** → **최신순** (`pendingGradeCount` BE 필드) |

---

## 12.10-3 · 채점 방식

| 유형 | 판별 | 입력 |
|------|------|------|
| 숙제 | 전체 | **맞은 수** (정답지 선택) |
| 수업기록 테스트 | `lesson_record_id IS NOT NULL` | **맞은 수** |
| 독립 테스트 (중간·기말) | `lesson_record_id IS NULL` | **틀린 문항 체크** (정답지 필요) |

별도 `exam_type` 규칙 없음.

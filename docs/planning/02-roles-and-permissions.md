# 02. 사용자 역할 및 권한 체계

## 역할 목록

| 그룹 | 역할 | 설명 |
|------|------|------|
| 관리자 | 원장·학원 관계자 | 전체 관리, 수강반 배정, 반 생성 |
| 선생님 | 국어/영어/수학 담임 | 담임 반 운영, 조교 반 지정 |
| 선생님 | 행정 | 학원 전체 보조, **학원 공지** 등록 ([DECISIONS.md](./DECISIONS.md) §2) |
| 조교 | 국어/영어/수학 조교 | 담당 반 한정 운영 |
| 학생 | 학생 | 수강 반 조회·신청·인증 |

교직원 회원가입 시 **역할 7종** 선택 → 원장 승인 시 역할·과목 **확정/수정** ([DECISIONS.md](./DECISIONS.md) §1, §2-1).

| 코드 | 역할 |
|------|------|
| `TEACHER_KO` / `TEACHER_EN` / `TEACHER_MATH` | 국·영·수 담임 |
| `STAFF_OFFICE` | 행정 |
| `ASSISTANT_KO` / `ASSISTANT_EN` / `ASSISTANT_MATH` | 국·영·수 조교 |

`ACADEMY_ADMIN`(원장·관계자)은 7종 외 — 승인 권한, 별도 계정.

## 관리자

- 전체 학생·반·선생님·조교 관리
- **수강반 배정**: 관리자만
- 반 생성·담임 지정
- 선생님 가입 시 역할·과목 지정

## 담임 선생님 (원본 2-2 표)

| 역할 | 과목 | 주요 권한 |
|------|------|-----------|
| 국어 선생님 | 국어 | 담임 반 커리큘럼 운영, 조교 담당 반 지정 |
| 영어 선생님 | 영어 | 담임 반 커리큘럼 운영, 조교 담당 반 지정 |
| 수학 선생님 | 수학 | 담임 반 커리큘럼 운영, 조교 담당 반 지정 |
| 행정 | – | 학원 전체 운영 보조, **공지 등록** |

## 담임 선생님 (요약)

- 담임 반 커리큘럼 운영
- 조교에게 **담당 반 지정**
- 클리닉 세션 생성·조교 배정
- 성실도 보고서 조회·코멘트·출력

## 조교 (원본 2-3 표)

| 역할 | 담당 반 지정 | 조교 가능 기능 |
|------|-------------|----------------|
| 국어 조교 | 담임 선생님이 지정 | 공지·숙제·테스트 등록 / 결과 입력 / 클리닉 신청 목록 확인 / 공부기록 열람 |
| 영어 조교 | 담임 선생님이 지정 | 동일 |
| 수학 조교 | 담임 선생님이 지정 | 동일 |

## 조교 (요약)

- 담임이 지정한 반만 접근
- 공지·숙제·테스트 등록 및 결과 입력
- 클리닉 신청 목록·결과 입력
- 공부기록 열람 (담당 반)

## 학생

- 가입 후 관리자 배정 전: **반 콘텐츠 열람 불가**
- 배정된 반만 조회
- **매주 월~금** 클리닉 시간 직접 신청, **토요일 23:00 이후** 변경 잠금
- 영상 수강 후 인증사진 촬영 또는 앨범 첨부
- 반 업데이트(**공지·숙제·테스트·클리닉 변경**) 시 앱 알림 수신

## RBAC 구현 방침

```
권한 = f(역할, academy_id, class_id, enrollment | assignment)
```

- API마다 `@PreAuthorize` + 서비스 레이어에서 `class_id` 소속 재검증
- 학생: `class_enrollment` 존재 여부
- 조교: `assistant_class_assignment` 존재 여부
- 담임: `class.homeroom_teacher_id = user.id`

자세한 기능별 표는 [03-feature-matrix.md](./03-feature-matrix.md).

# 05. 역할별 화면 설계

## 5.1 선생님(담임)

### 대시보드 (원본 5-1)

- 내가 담임인 반 목록 예: `[ 국어 남고1반 ]` `[ 국어 고1 여름방학 특강 ]` `[ 영어 남고2반 ]`
- 이번 주 주요 일정 예:
  - 수 18:00 국어 남고1반 수업
  - 금 20:00 테스트 마감 — 운수 좋은 날 5문항
- 미확인 알림 수 (예: 3)

### 반 상세 (편집 모드) — 탭

| 탭 | 기능 |
|----|------|
| 수업 정보 | 요일·시간·강의실 |
| 공지 | 작성·첨부·예약 발송 |
| 교재 | 제목·출판사·진도 |
| 영상 | URL → 썸네일 자동 |
| 숙제 | 예정 등록 / 결과 입력 |
| 테스트 | 예정 등록 / 결과 입력 |
| 클리닉 | 슬롯·조교·예약 현황 |
| 학생 목록 | 명단·출석·공부기록 |
| 보고서 | 성실도 보고서 조회·PDF |

### 조교 담당 반 지정 (원본 5-3)

- 조교 목록에서 담당 조교 선택 후 담당 반 체크
- 예: `이수민 조교 → [✓ 국어 남고1반] [✓ 국어 남고2반] [□ 영어 남고1반]`

## 5.2 조교

### 홈 (원본 6-1)

- 내가 담당하는 반 예: `[ 국어 남고1반 ]` `[ 국어 남고2반 ]`
- 이번 주 클리닉 예약 현황 예: `국어 남고1반 — 월 19:00 3명 / 4슬롯`

### 담당 반 기능 (원본 6-2)

- 공지 등록 — 반 공지 작성 및 게시
- 숙제 등록 — 예정 숙제 제목·마감일시 / 완료 결과 입력
- 테스트 등록 — 예정 테스트 제목·일시 / 완료 결과 입력
- 클리닉 신청 목록 확인 — 이번 주 학생별 예약 슬롯
- 클리닉 결과 입력 — 출석 여부 및 내용 메모
- 공부기록 열람 — 숙제·클리닉·테스트 달성률

### 클리닉 신청 목록 (원본 6-3)

표 컬럼: 요일 | 시간 | 학생명 | 예약 상태 | 클리닉 결과  
예: 월 19:00 김민준 예약됨 [입력 버튼] / 월 19:30 이서연 미예약 –

## 5.3 학생

### 메인 홈 (원본 7-1)

- `안녕하세요, {이름} 학생` + 🔔 알림 뱃지
- 수강 과목 선택: `[🟣 국어]` `[⬜ 수학 (미수강)]` `[🟢 영어]` — **색 있는 과목만 선택 가능**
- 과목별 수강반 목록 (예: 국어 — 남고1반, 고1 여름방학 고전소설 특강)

### 반 홈 (원본 7-2, 예: 국어 남고1반)

- 내 클리닉: `월 19:00 — 박지훈 조교`, `수 18:30 — 이수민 조교`
- `[ 공부 기록 ]` — 종합 72% 🟢, 게이지 `▓▓▓▓▓▓▓░░░`
- 수업 정보: `매주 화·목 18:00–20:30 / 301호`
- 최근 공지, 교재(진도), 영상(썸네일·**[인증사진 제출]**), 숙제(완료·예정), 테스트(완료·예정)
- 숙제 예: `완료: … [제출 완료]`, `예정: … ~5/28`

### 클리닉 예약 (원본 7-3)

- `이번 주 (5/26–5/30) | 토요일 23:00 전까지 변경 가능`
- 표: 요일 | 시간1 | 시간2 | 담당 조교 | 내 선택 | 상태

### 공부기록 상세 (원본 7-4)

- 숙제 제출률 / 클리닉 출석률 / 테스트 점수 / 영상 인증 — 항목별 %·게이지
- 종합 달성률 + 격려 문구 (예: 잘 하고 있어요! 🟢)

### 알림 목록

- 전체 / 미확인 필터

## 예약 규칙 (유지)

| 규칙 | 내용 |
|------|------|
| 예약 | 월 00:00 ~ 토 23:00 |
| 요일 | **월~금** |
| 잠금 | 토 23:00 |

- 수업기록 클리닉 등록 시 선택 → 종료 후 결과 입력

## Cron

- 월 00:00: 다음 주 슬롯
- 토 23:00: LOCK + 확정 알림
